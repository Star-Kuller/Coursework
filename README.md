# Курсовая работа 2025
Мой проект это веб-практикум для фреймворков, который позволяет практиковаться в работе с реальными фреймворками на практических задачах.
Подробнее можно узнать в [содержательной части курсовой](Курсовая%202025.docx)

## Дополнительно
Архитектура проекта была пожертвована в угоду скорости разработки. В архитектуре есть ряд грубейших ошибок, которые затрудняют дальшейшее развите проекта.
Развите этого проекта не предполагается. В случае возвращения к этой идеи я начну этот проект с 0 на других технологиях.
Не реализован Docker и Docker-compose файлы.

## Зависимости
- .NET 8.0
- PostgreSQL
